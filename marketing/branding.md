# Branding Kit — Agencia

## Paleta de colores

| Color | Hex | Uso |
|---|---|---|
| Fondo oscuro | `#0f172a` (slate-900) | Fondo principal |
| Fondo más oscuro | `#020617` (slate-950) | Nav, footer, secciones alternas |
| Primario | `#6366f1` (indigo-500) | Botones, CTAs, acentos |
| Primario hover | `#818cf8` (indigo-400) | Hover de botones |
| Secundario | `#06b6d4` (cyan-500) | Gradientes, detalles |
| Texto principal | `#ffffff` | Títulos |
| Texto secundario | `#94a3b8` (slate-400) | Párrafos |
| Borde | `rgba(255,255,255,0.1)` | Bordes sutiles |

## Tipografía

- **Títulos**: Geist Sans (de Vercel/Next.js)
- **Cuerpo**: Geist Sans
- **Monospace**: Geist Mono (para código)
- Tamaños base: 16px body, jerarquía de títulos 24-48px

## Tono de voz

- **Directo y profesional** — sin rodeos
- **Cercano pero formal** — tratamos de "tú"
- **Entusiasta con la tecnología** — mostramos pasión
- **Nunca vendemos, resolvemos** — el enfoque es el problema del cliente

## Logo conceptual

Letras "A" estilizada como un tag de programación: `<A/>`

Puedes generarlo gratis en:
- [Canva](https://canva.com) — buscar "logo tech"
- [Looka](https://looka.com) — generador con IA
- O pedirme que te genere un SVG

## Sugerencia para logo SVG minimalista

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 200 60" fill="none">
  <text x="10" y="42" font-family="monospace" font-size="36" font-weight="bold" fill="#6366f1">
    &lt;A/&gt;
  </text>
</svg>
```
